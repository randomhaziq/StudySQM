# Software Quality Control (SQC)

Software Quality Control (SQC) is one of the most important topics in Software Quality Management because it focuses on **detecting defects in software products** before they are delivered to customers.

---

# 1. What is Software Quality Control (SQC)?

Software Quality Control (SQC) is a set of activities used to verify that software products meet specified requirements and quality standards.

### Main Purpose

- Detect defects
- Verify correctness
- Validate functionality
- Ensure software quality
- Reduce customer-reported issues

### Key Idea

> Quality Control is product-oriented and focuses on finding defects in the software product.

---

# 2. Quality Assurance (QA) vs Quality Control (QC)

| Quality Assurance (QA) | Quality Control (QC) |
|------------------------|---------------------|
| Process-oriented | Product-oriented |
| Prevent defects | Detect defects |
| Proactive | Reactive |
| Focus on process improvement | Focus on product verification |
| Audits and standards | Testing and inspections |

### Example

#### QA Activities

- Establish coding standards
- Create development procedures
- Conduct process audits
- Define quality processes

#### QC Activities

- Perform testing
- Conduct inspections
- Verify outputs
- Detect defects

# 5. Software Testing

Testing is the most common Software Quality Control activity.

## Verification vs Validation

### Verification

**Question:** Are we building the product correctly?

Verification ensures that software artifacts comply with specifications.

### Examples

- Reviews
- Walkthroughs
- Inspections
- Static analysis

---

### Validation

**Question:** Are we building the right product?

Validation ensures the software satisfies user needs and requirements.

### Examples

- Functional testing
- System testing
- User Acceptance Testing (UAT)

---

# 7. Software Quality Metrics Used in QC

Metrics help evaluate software quality and testing effectiveness.

---

## Defect Density

Measures the number of defects relative to software size.

### Formula

```text
Defect Density = Number of Defects / KLOC
```

Where:

- KLOC = Thousand Lines of Code

---

## Test Coverage

Measures the percentage of software tested.

### Formula

```text
Test Coverage = Covered Code / Total Code × 100%
```

---

## Defect Removal Efficiency (DRE)

Measures the effectiveness of defect removal activities.

### Formula

```text
DRE = Defects Found Before Release / Total Defects × 100%
```

---

## Defect Leakage

Measures defects that escape into production.

### Formula

```text
Defect Leakage = Production Defects / Total Defects × 100%
```
