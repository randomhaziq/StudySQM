### Step 1: Go to Jira Sign-Up Page
- Visit: https://www.atlassian.com/software/jira
- Click “Get it free”

### Step 2: Create an Atlassian Account
- Sign up using:
    - Email + password
    - Or Google account
>This creates your Atlassian account (required for Jira)


### Step 3: Create Your Jira Site (Important Step)
You will be asked to create your site URL:

Format:
```
https://your-site-name.atlassian.net
```
Example:
```
https://agrokedah.atlassian.net
```
Choose wisely — this becomes your permanent Jira site address.


### Step 4: Name Your Organization
Enter your:
   - Organization name (e.g., AgroKedah)
   - Team size
   - Use case (e.g., Software Development, Project Management)

### Step 5: Choose a Template / Project Type
After site creation, Jira will prompt you to create your first project. Choose one:
- Scrum → Agile sprint-based projects
- Kanban → Continuous workflow (good for beginners)
- Bug tracking → Issue-focused tracking

### Step 6: Invite Team Members
- Add team members via email
- Or skip and invite later

### Step 7: Access Your Jira Site
Login at:
```
https://agrokedah.atlassian.net
```
You now have a fully functional Jira site
